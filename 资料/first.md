# 🚀 量化牛牛（QuantBull）

### —— AI 驱动的全链路量化投资智能体平台

本项目旨在构建一个集 **数据采集、AI 内容生成、量化策略、数字人视频、投资方案生成、前后端交互** 于一体的全自动金融智能平台。

系统包含：

* **量化牛牛主站（Web）**
* **量化牛牛微信小程序**
* **量化牛牛微信公众号 & 自动推送系统**
* **AI 中台（DeepSeek / Qwen / 内部微调模型）**
* **数据中台（行情、新闻、机构数据等）**
* **量化引擎（回测、策略、调仓、评分）**

# 🤖 AI 指令体系（Prompt 模板）

以下为平台内部实际使用的 **标准 Prompt 指令**（可直接复制到代码中）。

---

## **AI-01｜新闻生成指令模板（财联社快讯 → 深度新闻稿）**

```
你是一名专业金融编辑，请基于以下信息生成一篇 200-300 字的盘面分析：
【快讯】：{news}
【关联股票】：{code}
【行情数据】：{quote}

要求：
1. 内容客观、准确，不夸大风险
2. 包含投资者需关注的重点
3. 文末必须添加风险提示：“以上内容不构成投资建议，股市有风险，投资需谨慎。”
```

---

## **AI-02｜自然语言 → 量化策略代码生成（Backtrader）**

```
你是一名资深量化工程师，请将用户描述转换成 Backtrader 策略代码。

【策略描述】：
{user_text}

要求：
1. 生成可直接运行的 Python 代码
2. 必须包含 buy/sell 逻辑
3. 自动添加常用参数（周期、滑点、手续费）
4. 返回结果用 markdown 代码块格式
```

---

## **AI-03｜每日 AI 投资方案（3 ETF + 10 股票）**

```
你是智能投研系统，请基于市场数据生成《AI 投资内参》。

输入：
- 今日全市场行情数据
- ETF 板块资金流
- 新闻舆情分析结果
- LLM 对宏观事件的理解

输出：
1. 今日重点关注 3 只 ETF（附理由）
2. 今日看好的 10 只股票（附因子打分）
3. 简短市场综述
4. 风险提示（必须出现）
```

---

## **AI-04｜自动生成公司介绍（基于爬取数据）**

```
请根据以下公开数据为公司生成简介：
{company_data}

要求：
- 突出主营业务、优势、行业地位
- 不包含主观判断
- 120~250 字
```

---

## **AI-05｜数字人短视频脚本文案**

```
你现在是金融数字人主播，请为短视频生成口播稿。

输入：
{content}

要求：
- 45-65 秒
- 语言自然、口语化
- 情绪稳重
- 不构成投资建议
```

# 🚀 部署流程

### **1. 构建服务**

```bash
docker-compose build
```

### **2. 启动服务**

```bash
docker-compose up -d
```

### **3. 配置 Nginx**

用于 Web、小程序、公众号回调。

### **4. 配置定时任务**

Celery Beat / Crontab

* 每分钟：行情
* 实时：快讯
* 收盘后：AI 投资日刊
