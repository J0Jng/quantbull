# AI Generated Code

AI生成的代码仓库，用于存储和管理AI自动生成的代码片段、策略等。

## 目录结构

```
ai-generated-code/
├── README.md                # 说明文档
├── strategies/              # AI生成的策略代码
│   ├── technical/          # 技术策略
│   ├── factor/             # 因子策略
│   └── etf/                # ETF策略
├── scripts/                 # AI生成的脚本
│   ├── data_processing/    # 数据处理脚本
│   └── analysis/           # 分析脚本
├── templates/               # 代码模板
│   └── strategy_template.py
└── reviews/                 # 代码评审记录
    └── README.md
```
