# AI Prompts

AI提示词模板库，用于统一管理项目中使用的各类Prompt模板。

## 目录结构

```
ai-prompts/
├── README.md                # 提示词索引说明
├── news/                    # 新闻相关提示词
│   ├── news_generation.md  # 新闻生成
│   └── news_analysis.md    # 新闻分析
├── strategy/                # 策略相关提示词
│   ├── strategy_generation.md  # 策略代码生成
│   └── strategy_analysis.md    # 策略分析
├── content/                 # 内容相关提示词
│   ├── daily_report.md     # 投资日刊生成
│   ├── company_intro.md    # 公司介绍生成
│   └── video_script.md     # 视频脚本生成
├── analysis/                # 分析相关提示词
│   ├── market_analysis.md  # 市场分析
│   └── sentiment_analysis.md # 情绪分析
└── templates/               # 提示词模板
    └── base_template.md
```
